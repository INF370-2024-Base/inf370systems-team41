search Search Employees

This section provides assistance on how to search for employees within the system.

Common Questions

  • How can I search for an employee?

    Use the search bar at the top of the employee management page. Enter the name or contact details of the employee you are looking for, and the list will filter automatically.

Technical Issues

  • Why is my search not returning any results?

    Ensure that the search term is entered correctly. If the problem persists, it could be due to a database issue. Contact technical support for assistance.

Step-by-Step Guide

Search Employees
  1. Navigate to the 'Employees' page.
  2. Locate the search bar at the top of the page, next to the 'Add Employee' button.
  3. Enter the name or contact details of the employee you are looking for.
  4. The list of employees will filter in real-time based on your input.
  5. If the desired employee appears, you can edit or delete the record using the corresponding buttons.

person_add Add Employee

This section provides assistance on how to add a new employee within the system.

Common Questions

  • How do I add a new employee?

    To add a new employee, navigate to the 'Add Employee' page by clicking the 'Add Employee' button on the top right corner of the employee management page. Fill in the required fields and click 'Add Employee'.

  • What information is required to add an employee?

    The required fields are First Name, Last Name, and Contact Details. The Address is optional but recommended.

Technical Issues

  • What happens if I don't fill in all required fields?

    The form will not be submitted, and an error message will be displayed below each missing field indicating that it is required.

  • What if I encounter an error while adding an employee?

    If you encounter an error, check if all required fields are filled correctly. If the problem persists, contact technical support for further assistance.

Step-by-Step Guide

Add Employee
  1. Navigate to the 'Employees' page.
  2. Click the 'Add Employee' button located at the top right corner of the page.
  3. You will be redirected to the 'Add New Employee' form.
  4. Fill in the required fields:
    • First Name: Enter the employee's first name.
    • Last Name: Enter the employee's last name.
    • Contact Details: Provide the employee's contact information.
    • Address: Enter the employee's address (optional).
  5. Ensure all required fields are filled correctly.
  6. Click the 'Add Employee' button at the bottom of the form to submit.
  7. If the form is valid, the employee will be added, and you will see a confirmation message.
  8. You can now see the new employee listed in the employee management page.

edit Edit Employee

This section provides assistance on how to edit existing employee details within the system.

Common Questions

  • How do I edit an employee's information?

    To edit an employee's information, navigate to the 'Edit Employee' page by clicking the 'Edit Employee' button in the navigation menu or on the employee details page. Modify the relevant fields with the updated information and click 'Update' to save the changes. A confirmation message will appear if the employee's information is successfully updated.

  • What details can I edit?

    You can edit fields such as name, contact details, department, and position. Ensure all changes are accurate before saving to avoid any discrepancies in the employee's records.

Technical Issues

  • Why am I unable to save my edits?

    Check if all required fields are filled out correctly. Error messages will highlight any missing or incorrect information. Ensure there are no network issues and try refreshing the page. If the problem persists, contact technical support for assistance.

  • Why are my edits not reflecting in the employee's profile?

    There may be a delay in updating the employee's profile. Verify the changes were saved by checking for a confirmation message. If the changes still do not appear, contact technical support to investigate the issue further.

Step-by-Step Guide

Step 1: Access the 'Edit Employee' Page

Click on the 'Edit Employee' button in the navigation menu or on the employee details page.

 Step 2: Modify Employee Details

Update the relevant fields with the new information.

 Step 3: Save Changes

Click the 'Update' button to save the edits. Ensure all changes are accurate.

 Step 4: Confirm Update

Check for a confirmation message indicating successful update of the employee's information.

delete Delete Employee

This section provides assistance on how to delete an existing employee from the system.

Common Questions

  • How do I delete an employee?

    To delete an employee, navigate to the 'Delete Employee' page by clicking the 'Delete Employee' button in the navigation menu or on the employee details page. Confirm the deletion by clicking 'Delete' in the confirmation dialog. A confirmation message will appear if the employee is successfully deleted.

  • Can deleted employees be recovered?

    No, deleted employees cannot be recovered once deleted from the system. Ensure you have selected the correct employee for deletion before confirming.

Technical Issues

  • Why am I unable to delete an employee?

    Check if you have the necessary permissions to delete employees. Some user roles may not have the authorization to perform deletions. Contact your administrator if you believe this is an error.

  • Why is the delete confirmation dialog not appearing?

    This could be due to a browser issue or script blocker. Ensure pop-ups are enabled for the site and try refreshing the page. If the problem persists, contact technical support for assistance.

Step-by-Step Guide

Step 1: Access the 'Delete Employee' Page

Click on the 'Delete Employee' button in the navigation menu or on the employee details page.

 Step 2: Confirm Deletion

Review the employee's details to ensure the correct employee is selected for deletion.

 Step 3: Click 'Delete'

Click the 'Delete' button in the confirmation dialog to proceed with deletion.

 Step 4: Confirm Deletion

Check for a confirmation message indicating successful deletion of the employee from the system.

schedule Add Employee Daily Hours

This section provides assistance on how to add daily work hours for an employee within the system.

Common Questions

  • How do I add daily hours for an employee?

    To add daily hours for an employee, navigate to the 'Add Employee Daily Hours' page by clicking the 'Add Employee Daily Hours' button on the employee management page. Fill in the required fields and click 'Add Hours'.

  • What information is required to add daily hours?

    The required fields are Employee Name, Date, Start Time, End Time, and Description. The Date and Time fields are mandatory to track the employee's work hours effectively.

Technical Issues

  • What happens if I don't fill in all required fields?

    The form will not be submitted, and an error message will be displayed below each missing field indicating that it is required.

  • What if I encounter an error while adding daily hours?

    If you encounter an error, check if all required fields are filled correctly. If the problem persists, contact technical support for further assistance.

Step-by-Step Guide

Add Employee Daily Hours
  1. Navigate to the 'Employee Daily Hours' page.
  2. Click the 'Add Hours' button located at the top right corner of the page.
  3. You will be redirected to the 'Add New Daily Hours' form.
  4. Fill in the required fields:
    • Employee Name: Select the employee from the dropdown list.
    • Date: Enter the date for which you want to add hours.
    • Start Time: Enter the start time of the work period.
    • End Time: Enter the end time of the work period.
    • Description: Provide a brief description of the work performed (optional).
  5. Ensure all required fields are filled correctly.
  6. Click the 'Add Hours' button at the bottom of the form to submit.
  7. If the form is valid, the daily hours will be added, and you will see a confirmation message.
  8. You can now see the added daily hours listed in the employee management page.
