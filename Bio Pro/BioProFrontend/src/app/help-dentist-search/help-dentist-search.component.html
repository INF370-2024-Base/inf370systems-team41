Help: Search Dentist

Description:
The search functionality allows you to find dentists in the system by their first name, last name, or contact details.

How to Use:
Enter the search criteria (first name, last name, or contact details) in the search input field.
Click the "Search" button or type in the search box to filter the list of dentists.

Required Inputs:
First Name
Last Name
Contact Details

Restrictions:
The search is case-insensitive.
If the search query is empty, all dentists will be displayed.

search Search Dentist

Example:

To find a dentist named "John Doe", enter "John", "Doe", or "John Doe" in the search field and click "Search".
