Help: Write Off Stock

Description:
The write off stock functionality allows you to remove damaged or unusable stock from the system.

How to Use:
Navigate to the write off stock page.
Select the stock item to be written off.
Enter the quantity, date, and reason for the write off.
Click "Submit" to remove the stock from the system.

Required Inputs:
Stock Item, Quantity Written Off, Written Off Date, Reason.

Example:

To write off stock, navigate to the write off stock page, select the stock item, enter the quantity, date, and reason, and click "Submit".

FAQ:

Q: How do I write off stock?
A: Navigate to the write off stock page, select the stock item, enter the quantity, date, and reason, and click "Submit".

Q: What information do I need to write off stock?
A: You need to provide the Stock Item, Quantity Written Off, Written Off Date, and Reason.

Q: Why can't I submit the form?
A: Ensure all required fields are filled out correctly. The "Submit" button will be enabled only if the form is valid.

Q: How do I know if the stock was written off successfully?
A: Upon successful write off, a confirmation message will be displayed. If there is an error, an error message will appear.

Q: What should I do if I encounter an error while writing off stock?
A: Check the error message for details and ensure all required fields are correctly filled. If the issue persists, contact support.
