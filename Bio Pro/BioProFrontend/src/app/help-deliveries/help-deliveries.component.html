local_shipping Deliveries

Overview

The Deliveries page provides a comprehensive view of all the deliveries made by employees. This page displays essential information about each delivery, including the delivery ID, date, status, associated employee, and the system order ID.

Page Elements {{ dropdownStates['pageElements'] ? 'expand_less' : 'expand_more' }}
  • Delivery ID: A unique identifier for each delivery.
  • Delivery Date: The date when the delivery was made.
  • Delivery Status: The current status of the delivery (e.g., pending, delivered).
  • Employee: The employee who made the delivery.
  • System Order ID: The associated system order ID for the delivery.
Common Questions {{ dropdownStates['commonQuestions'] ? 'expand_less' : 'expand_more' }}

How do I view the details of a delivery?

The details of each delivery are displayed within individual cards on the Deliveries page. Simply scroll through the page to view all deliveries.

What information is displayed for each delivery?

Each delivery card displays the Delivery ID, Delivery Date (if available), Delivery Status, Employee details, and System Order ID.

How can I tell which employee made a particular delivery?

The employee's first and last name are displayed on each delivery card under the Employee section.

What do the different delivery statuses mean?

The delivery statuses provide information on the current state of the delivery. Common statuses include pending (the delivery is yet to be made), and delivered (the delivery has been successfully completed).

Why is the delivery date not showing for some deliveries?

The delivery date may not be available for all deliveries. If the delivery date is not set, it will not be displayed on the delivery card.

Can I search for specific deliveries?

Currently, the Deliveries page does not support search functionality. You can manually scroll through the list to find specific deliveries.

How often is the delivery information updated?

The delivery information is updated in real-time. Any changes made to deliveries are immediately reflected on the Deliveries page.

What should I do if the delivery details are incorrect?

If you notice any incorrect details, please contact the support team to have the information reviewed and corrected.

Is it possible to filter deliveries by status?

Currently, the Deliveries page does not have a filtering feature. You can manually check each delivery's status.

Can I export the delivery data?

At the moment, exporting delivery data is not supported. If you need to export data, please contact the support team for assistance.

Why are some delivery details not visible?

Some delivery details might not be visible due to missing or incomplete data. Ensure that all required fields are filled out when creating a delivery.

Overview

The Add New Dentist page allows you to add a new dentist to the system. This form requires information such as the dentist's surname, first name, contact details, and address.

Page Elements {{ dropdownStates['pageElements'] ? 'expand_less' : 'expand_more' }}
  • Surname: The last name of the dentist. This field is required.
  • Name: The first name of the dentist. This field is required.
  • Contact Details: The dentist's contact information. This field is required.
  • Address: The dentist's address. This field is required.
  • Add Dentist Button: Submits the form to add the dentist.
  • Cancel Button: Cancels the action and navigates back to the list of dentists.
Common Questions {{ dropdownStates['commonQuestions'] ? 'expand_less' : 'expand_more' }}

How do I add a new dentist?

Fill in all the required fields (Surname, Name, Contact Details, and Address) and click the "Add Dentist" button.

What happens if I don't fill in all the required fields?

If any required fields are left blank, the form will display error messages, and you will not be able to submit the form until all required fields are filled out.

How do I cancel adding a new dentist?

Click the "Cancel" button to stop the action and navigate back to the list of dentists.

What should I do if I receive an error message when submitting the form?

If you receive an error message, it could be due to a network issue or a problem with the backend service. Ensure your internet connection is stable and try again. If the issue persists, contact support.

How can I tell if the dentist was added successfully?

If the dentist is added successfully, you will be redirected to the list of dentists, and a success message will appear. If there's an issue, an error message will be displayed.

Can I edit the information after adding a dentist?

Currently, you cannot edit a dentist's information directly after adding them. You would need to contact support for any modifications.

Why is the contact detail format important?

Correct formatting of contact details ensures that communication with the dentist is effective and reliable. Follow the specified format to avoid errors.

How can I ensure the address is entered correctly?

Make sure to provide a complete and accurate address to facilitate proper record-keeping and correspondence.

Is there a limit to the number of dentists I can add?

There is no specified limit to the number of dentists you can add. Ensure that each entry is accurate and complete.
