Help: Add Dentist

Description:
The add functionality allows you to create a new dentist record in the system.

How to Use:
Click on the "Add Dentist" button to open the add dentist form.
Fill in the required fields (Last Name, First Name, Contact Details, Address).
Click the "Add Dentist" button to save the new dentist.

Required Inputs:
Last Name (required)
First Name (required)
Contact Details (required)
Address (required)

Restrictions:
All fields are mandatory.
Ensure that the contact details are valid.

person Surname person Name phone Contact Details location_on Address

Example:

Fill in the form with the dentist's surname, name, contact details, and address, then click "Add Dentist".
