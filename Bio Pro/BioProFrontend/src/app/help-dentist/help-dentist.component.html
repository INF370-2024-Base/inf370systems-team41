search Search Dentist

This section provides assistance on how to search for dentists within the system.

Common Questions

  • How can I search for a dentist?

    Use the search bar at the top of the dentist management page. Enter the name or contact details of the dentist you are looking for, and the list will filter automatically.

Technical Issues

  • Why is my search not returning any results?

    Ensure that the search term is entered correctly. If the problem persists, it could be due to a database issue. Contact technical support for assistance.

Step-by-Step Guide

Search Dentist
  1. Navigate to the 'Dentists' page.
  2. Locate the search bar at the top of the page, next to the 'Add Dentist' button.
  3. Enter the name or contact details of the dentist you are looking for.
  4. The list of dentists will filter in real-time based on your input.
  5. If the desired dentist appears, you can edit or delete the record using the corresponding buttons.

person_add Add Dentist

This section provides assistance on how to add a new dentist within the system.

Common Questions

  • How do I add a new dentist?

    To add a new dentist, navigate to the 'Add Dentist' page by clicking the 'Add Dentist' button on the top right corner of the dentist management page. Fill in the required fields and click 'Add Dentist'.

  • What information is required to add a dentist?

    The required fields are First Name, Last Name, and Contact Details. The Address is optional but recommended.

Technical Issues

  • What happens if I don't fill in all required fields?

    The form will not be submitted, and an error message will be displayed below each missing field indicating that it is required.

  • What if I encounter an error while adding a dentist?

    If you encounter an error, check if all required fields are filled correctly. If the problem persists, contact technical support for further assistance.

Step-by-Step Guide

Add Dentist
  1. Navigate to the 'Dentists' page.
  2. Click the 'Add Dentist' button located at the top right corner of the page.
  3. You will be redirected to the 'Add New Dentist' form.
  4. Fill in the required fields:
    • Surname: Enter the dentist's surname.
    • Name: Enter the dentist's first name.
    • Contact Details: Provide the dentist's contact information.
    • Address: Enter the dentist's address (optional).
  5. Ensure all required fields are filled correctly.
  6. Click the 'Add Dentist' button at the bottom of the form to submit.
  7. If the form is valid, the dentist will be added, and you will see a confirmation message.
  8. You can now see the new dentist listed in the dentist management page.

edit Edit Dentist

This section provides assistance on how to edit existing dentist details within the system.

Common Questions

  • How do I edit a dentist's information?

    To edit a dentist's information, navigate to the 'Edit Dentist' page by clicking the 'Edit Dentist' button in the navigation menu or on the dentist details page. Modify the relevant fields with the updated information and click 'Update' to save the changes. A confirmation message will appear if the dentist's information is successfully updated.

  • What details can I edit?

    You can edit fields such as name, contact details, specialization, and availability. Ensure all changes are accurate before saving to avoid any discrepancies in the dentist's records.

Technical Issues

  • Why am I unable to save my edits?

    Check if all required fields are filled out correctly. Error messages will highlight any missing or incorrect information. Ensure there are no network issues and try refreshing the page. If the problem persists, contact technical support for assistance.

  • Why are my edits not reflecting in the dentist's profile?

    There may be a delay in updating the dentist's profile. Verify the changes were saved by checking for a confirmation message. If the changes still do not appear, contact technical support to investigate the issue further.

Step-by-Step Guide

Step 1: Access the 'Edit Dentist' Page

Click on the 'Edit Dentist' button in the navigation menu or on the dentist details page.

 Step 2: Modify Dentist Details

Update the relevant fields with the new information.

 Step 3: Save Changes

Click the 'Update' button to save the edits. Ensure all changes are accurate.

 Step 4: Confirm Update

Check for a confirmation message indicating successful update of the dentist's information.

delete Delete Dentist

This section provides assistance on how to delete an existing dentist from the system.

Common Questions

  • How do I delete a dentist?

    To delete a dentist, navigate to the 'Delete Dentist' page by clicking the 'Delete Dentist' button in the navigation menu or on the dentist details page. Confirm the deletion by clicking 'Delete' in the confirmation dialog. A confirmation message will appear if the dentist is successfully deleted.

  • Can deleted dentists be recovered?

    No, deleted dentists cannot be recovered once deleted from the system. Ensure you have selected the correct dentist for deletion before confirming.

Technical Issues

  • Why am I unable to delete a dentist?

    Check if you have the necessary permissions to delete dentists. Some user roles may not have the authorization to perform deletions. Contact your administrator if you believe this is an error.

  • Why is the delete confirmation dialog not appearing?

    This could be due to a browser issue or script blocker. Ensure pop-ups are enabled for the site and try refreshing the page. If the problem persists, contact technical support for assistance.

Step-by-Step Guide

Step 1: Access the 'Delete Dentist' Page

Click on the 'Delete Dentist' button in the navigation menu or on the dentist details page.

 Step 2: Confirm Deletion

Review the dentist's details to ensure the correct dentist is selected for deletion.

 Step 3: Click 'Delete'

Click the 'Delete' button in the confirmation dialog to proceed with deletion.

 Step 4: Confirm Deletion

Check for a confirmation message indicating successful deletion of the dentist from the system.
