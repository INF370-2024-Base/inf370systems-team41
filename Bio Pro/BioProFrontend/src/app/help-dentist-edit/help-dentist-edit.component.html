Help: Edit Dentist

Description:
The edit functionality allows you to update the details of an existing dentist.

How to Use:
Click on the "Edit" button next to the dentist you want to edit.
Modify the required fields (Last Name, First Name, Contact Details, Address).
Click the "Save" button to apply the changes.

Required Inputs:
Last Name (required)
First Name (required)
Contact Details (required)
Address (optional)

Restrictions:
All fields except Address are mandatory.
Ensure that the contact details are valid.

First Name Last Name Contact Details Address

Example:

Modify the details in the form and click "Save" to update the dentist's information.
